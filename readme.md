## Installation and usage

1. Clone this repository and run `npm i`.

2. Run `npm start` to start the development server.

## Tech Stack

The application was built using the following technologies:

-   React.js
-   TypeScript
-   Redux Toolkit / RTK Query
-   Chakra UI
-   Styled-components
-   React Router Dom v6
-   Vite
-   React hook form
-   Yup
